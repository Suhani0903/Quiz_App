# Quiz_Application
This is a simple yet interactive Quiz Application built using Java Swing that allows users to test their knowledge.The application provides an engaging user interface with multiple features like login,rules page, quiz gameplay,and a score summary.

**Features**
Login Page: Users are required to log in with their credentials (username and password) before accessing the quiz. This adds a personalized touch and ensures a more controlled experience.
Quiz Page: The main feature of the application, where users can answer multiple-choice questions within a set time limit. Questions are randomly selected to provide an engaging and unpredictable quiz experience.
Rules Page: A page dedicated to explaining the rules of the quiz, such as how the quiz works, the time limit for each question, and the scoring system.
Score Page: After the quiz is completed, users are presented with a score page showing their results and the total score they achieved. This allows users to track their performance and review their results.

**Technologies Used**
Java: The core programming language used to build the application.
Swing: Java Swing is used for building the graphical user interface (GUI), providing a sleek and intuitive user experience.

Run the application. The main entry point will open the login page, where you can enter your credentials to start the quiz.

**How to Play**
Log in: Enter your username on the login screen to access the quiz.
Review the Rules: If you're unsure about the game mechanics, refer to the rules page for detailed instructions on how the quiz works.
Start the Quiz: After logging in, you'll be redirected to the quiz page. Answer multiple-choice questions, and the timer will keep track of the time for each question
View Your Score: Once the quiz is completed, the application will display your score on the results page.
